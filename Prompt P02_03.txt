buatlahh sebuah program sistem manajemen kafetariia di kampus Institut Teknologi Del menggunakan bahasa c yang dapat dikompilasi mengguunakan GCC MinGW64 dan dijalankan melalu terminal powershelll atau command prompt Windows, program ini harus mensimulasikan operasional sederhana kafetaria ITDEL , dengan didalamnya ada beberapa fitur yang saya inginkan seperti menu utama program yang dimana didalamnya ada daftar menu, pesan makanan/minuman, lihat total pesanan, pembayaran , dan keluar. Lau pada daftar menu ada nama menu seperti pisang kulit lumpiah, pisang coklat, Roti isi ayam, Cappucino, Es Teh . Yang dimana harganya pisang kulit lumpiah 10.000, pisang coklat 10.000, roti isi ayam 15.000, cappuciino 8.000, dan es teh 5.000. Menu tersebut ditunjukkan menggunakan Array of struct. Dalam proses pemesanan user dapat memasukkan makanan yang dia inginkan lalu memasukkan jumlah pesanannya. Program harus bisa memvalidasi  apakah menu  tersebut ada, menghitung total yaitu harga tambah jumlah pesanan, lalu menyimpan pesanan dalam Array Off Struct. Lalu programm harus menampilkan total pesanan dalam daftar pesanan. Program harus pakai struct, array, looping, if else, function, input output
